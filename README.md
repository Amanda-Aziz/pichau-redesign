# 🛒 Pichau Clone - E-commerce Interface

Este projeto é uma recriação da interface do site da [Pichau Informática](https://pichau.com.br), desenvolvido com fins puramente educacionais para praticar habilidades de desenvolvimento frontend e UI/UX.

> **🚀 Status do Projeto:** [Concluído]

---

## 💻 Sobre o Projeto

O objetivo principal deste repositório é demonstrar a capacidade de reconstruir uma plataforma de e-commerce complexa, focando em **responsividade**, **componentização** e **performance**.

A ideia foi replicar a experiência do usuário da Pichau, desde a listagem de componentes de hardware até a visualização detalhada de produtos.

## ✨ Funcionalidades Implementadas

- [x] Header Funcional: Barra de busca centralizada, menu de conta/favoritos e contador de itens no carrinho.
- [x] Menu de Navegação Estruturado: Barra de categorias destacada com links para Departamentos, Monte seu PC, PCs Gamer e Promoções.
- [x] Vitrine de Produtos (Grid): Exibição organizada de produtos com imagens centralizadas e títulos detalhados.
- [x] Lógica de Preços e Descontos: Implementação visual de preço antigo (riscado), preço atual em destaque e cálculo de parcelamento/desconto no PIX.
- [x] Call to Action (CTA): Botões de "Comprar" padronizados em todos os cards de produtos.
- [x] UI Temática (Dark Mode): Aplicação da paleta de cores oficial (Preto e Vermelho) para máxima fidelidade visual.
- [ ] Interatividade do Carrinho: Lógica para adicionar itens e atualizar o contador do header.
- [ ] Filtros Avançados: Menu lateral para refinar buscas por preço e marca.
